Get All Tasks

caution

By now, Completed tasks are not implemented in the REST API, so no completed task will be returned by using this client.

getAll()#

const myClient // an authorized client instance
myClient.task.getAll().then(res => console.log(res)) // ["task1 content", "task2 content", ...]

getAllJSON()#

const myClient // an authorized client instance
myClient.task.getAllJSON().then(res => console.log(res)) // [{task1...}, {task2...}, ...]
